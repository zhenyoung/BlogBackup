## 1.3 导入图片

通过 Ctrl+Shift+I （也可输入 \![]() 来实现）然后从文件目录中添加图片，如下

<img src="https://pic-blogs.oss-cn-beijing.aliyuncs.com/img/Penguim.jpg"  />

也可直接拖拽，也可复制粘贴过来，如下

![Penguim](https://pic-blogs.oss-cn-beijing.aliyuncs.com/img/Penguim.jpg)

注意：对于本地图片，若已经删除了（即在电脑当前目录下无法找到图片），则导出的文件中就无法显示图片；故而发送给其他用户的typora文档，图片是无法显示的，因为对于其他用户而言是不会在我的图片路径下找到相应的图片，然而对于从网络上粘贴的图片，接收的用户则可以查看，在断网的情况下依然可以显示

## 1.4 添加超链接

## 1.5 列表和表格

### 1.5.1 列表

#### 1.5.1.1 无序列表

在单独的一行输入 *+空格，或者 -空格，或者++空格，如下

* 这是用*创建的列表

- 这是用-创建的列表

+ 这是用+创建的列表

* 无序列表1
* 无序列表2（在列表后输入 Tab 即可降级，Shift + Tab 降级，如下）
  * 无序列表3
    * 无序列表4
      * 有序列表5

也可使用Ctrl+Shift+]  对选中的文本所在段落生成无序列表

#### 1.5.1.2 有序列表

在单独的一行输入 数字+.+空格，如下

1. 有序列表1
2. 有序列表2（在列表后输入 Tab 即可降级，Shift + Tab 降级，如下）
   1. 有序列表3
      1. 有序列表4
         1. 有序列表5

也可使用 Ctrl+Shift+[  对选中的文本所在段落生成有序列表

#### 1.5.1.3 任务列表

在单独一行输入	\-空格[空格]空格（在列表后输入 Tab 即可降级，Shift + Tab 降级，如下）

- [x] 吃饭
  - [x] 早上吃啥
  - [x] 中午吃啥
  - [x] 晚上吃啥
- [x] 睡觉
  - [ ] 早上几点起
  - [x] 晚上几点睡
- [ ] 打豆豆
  - [x] 我是一个无情的点赞机器

综上，Typora中，列表均可通过Tab和Shift+Tab来对列表进行升级和降级操作

## 1.7 脚注

类似书中对一些内容作一些详尽的解释，格式：脚注\[^数字]，如下

饮中八仙歌[^1]

尼采[^2]

脚注[^3]

注释[^4]

一般把脚注写在文章末尾

## 1.9 行内代码和代码块

### 1.9.1 行内代码

对于简单的一句代码或者需要强调显示的文本，格式：\`code or text\`（或者选中内容，Ctrl+Shift+`），如下

`printf("笨鸟先飞");`

`笨鸟先飞`

### 1.9.2 代码块

\```+编程语言（或者Ctrl+Shift+K），比如C和JAVA（若没有补加语说明，则不会显示高亮），然后在代码块中码源码，如下

```c
#include<stdio.h>
int main(){
    printf("Hello, World!!!");
    return 0;
}
```



```JAVA
public class HelloWorld{
  public static void main(String[] args){
    System.out.println("Hello, World!!!");
      return 0;
  }
}
```
